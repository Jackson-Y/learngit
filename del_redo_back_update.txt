1、更新文件
	a. 查看当前版本库状态，哪些文件被修改过，还没有准备提交的修改。
		git status
	b. 对比修改前后的内容变化
		git diff readme.txt
	c. 添加到版本库
		git add readme.txt
	d. 再次查看状态
		git status
	e. 提交
		git commit -m "修改readme.txt"
	f. 再次查看状态
		git status
